# RaidRecord

파이널 판타지 14 캐릭터의 레이드 클리어 이력을 한눈에 확인할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 레이드 클리어 이력 조회
- 클리어 주차, 시간, 직업, 파티 멤버 정보 확인
- 올스타 점수 및 순위 표시

## 사용 방법

### 1. FFLogs API 설정

이 프로그램을 사용하려면 FFLogs API 자격 증명이 필요합니다.

1. [FFLogs](https://www.fflogs.com)에 로그인
2. 우측 상단 프로필 → `Settings` → `Web API` 탭으로 이동
3. [Manage your V2 clients here](https://www.fflogs.com/api/clients/) 클릭
4. `Create Client` 버튼 클릭
4. 이름과 URL을 입력하고 Public 체크 해제 상태로 생성
5. 생성된 `Client ID`와 `Client Secret`을 프로그램에 입력

### 2. 레이드 선택

상단의 `레이드 선택` 버튼으로 검색할 레이드를 선택할 수 있습니다.
확장팩 이름을 클릭하면 해당 확장팩의 레이드를 일괄 선택/해제합니다.

### 3. 캐릭터 검색

검색창에 다음 형식으로 입력:
- `캐릭터명` (예: `빛의전사`) - 서버를 선택하는 화면이 표시됩니다
- `캐릭터명@서버` (예: `빛의전사@카벙클`, `빛의전사@Carbuncle`)
- `캐릭터명 서버` (예: `빛의전사 카벙클`, `빛의전사 Carbuncle`)

## 주의 사항

- FFLogs API는 시간당 요청 제한이 있습니다. 검색 실행시 현재 사용량을 확인할 수 있습니다.
- 모든 정보는 FFLogs 공개 리포트 기준으로 조사되며, 실제 클리어 정보와 다를 수 있습니다.
- 화요일 17시~19시 사이 클리어는 주차 판정이 모호할 수 있습니다.

## 기타 사항

- 한국 서버만 지원합니다, 글로벌은 툼스톤이 있으니 필요 없을거라 생각합니다.
- 핵심 로직 일부를 제외한 대부분의 코드는 Claude Code를 통한 바이브 코딩으로 작성되었습니다.
