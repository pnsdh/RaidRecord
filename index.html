FFLogs API 설정

📋 API 키 발급 방법

  1. FFLogs 계정이 없다면 FFLogs.com에서 먼저 계정을 생성합니다.
  2. API Clients 페이지로 이동합니다.
  3. "Create Client" 버튼을 클릭합니다.
  4. 다음 정보를 입력합니다:
    • Name: 원하는 이름 (예: "RaidRecord App")
    • Redirect URLs: 아무 URL이나 입력 (예: "http://localhost")
    → http:// 또는 https:// 를 반드시 포함해야 합니다
    • Public Client: 체크하지 마세요
  5. "Create" 버튼을 클릭하여 클라이언트를 생성합니다.
  6. 생성된 클라이언트의 Client ID와 Client Secret을 복사하여 위 입력란에 붙여넣습니다.
  7. "저장" 버튼을 클릭하면 자동으로 인증 토큰이 발급됩니다.
